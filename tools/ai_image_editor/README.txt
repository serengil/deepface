# AI Image Editor (DeepFace Tool)

A Gradio-based interface for:
- Image processing (detect faces, resize, rescale, rotate, masked image, smart blur, remove objects)
- Face verification (photo upload or webcam)
- Face analysis (age, gender, emotion, race)
- **Optional** chatbot assistant powered by OpenAI API

## Installation

This tool is **optional** in DeepFace.  
To install dependencies:

```bash
pip install -r deepface/tools/ai_image_editor/requirements.txt
